# foodDApp
A Decentralized crypto app that enables villagers to exchange foodstuffs using NFT

# Tech Stack
```
Solidity
Non-fungible Token (NFT)
RSK 
Metamask Wallet
React
OpenZeppelin (OZ)
```

To run:
```
npm install
truffle migrate --network testnet
cd frontend
npm start

```
